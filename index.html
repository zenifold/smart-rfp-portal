About Lead Consultant

Building Better Communities, LLC respectfully submits this proposal in response to SMART's Request for Proposals for an Enterprise Resource Planning (ERP) System Upgrade Consultant. Our proposal is led by Maximilian Murphy, who brings 15+ years of hands-on ERP migration experience combining strategic advisory with technical implementation. Throughout his consulting career, he has architected and deployed enterprise-grade data products, analytics platforms, and automated workflows serving organizations with 500+ users.

Our approach integrates human-centered design methodology with quantitative data analysis to ensure that technology implementations align with how people actually work while delivering measurable business outcomes through data-driven insights.

24-Month Implementation Roadmap

Click any phase to view detailed deliverables and milestones

1

Discovery & Assessment

Months 1-3

Comprehensive needs assessment across 8 departments, 20+ stakeholder interviews, current systems evaluation, technical infrastructure assessment, and integration requirements matrix.

Key Deliverables:

  • Project Charter and Governance Framework
  • Needs Assessment Report by functional area
  • As-Is vs. To-Be Process Documentation
  • Service Blueprints for major processes
  • Technical Infrastructure Assessment
  • Integration Requirements Matrix
  • Requirements Traceability Matrix
Investment: 10% of total ($162,500)
2

Vendor Selection & Planning

Months 4-6

ERP RFP development with user stories, technical requirements definition, vendor identification, proposal evaluation support, reference checks, and finalist selection.

Key Deliverables:

  • Complete ERP RFP Document with User Stories
  • Technical Requirements Specification
  • Integration Requirements Specification
  • Vendor Evaluation Scoring Matrix
  • Functional Comparison Report
  • Reference Check Summary
  • Vendor Selection Recommendation Report
Investment: 29% of total ($565,000)
3

Implementation & Training

Months 7-18

Full-time implementation oversight including data migration hygiene, configuration documentation, Day-in-the-Life testing coordination, integration "Handshake" testing, and post-implementation stabilization.

Key Deliverables:

  • Bi-weekly Status Reports
  • Project Dashboard Updates
  • Risk Register Updates
  • Decision Logs
  • Monthly Executive Summary Reports
  • Configuration Documentation
  • Integration Testing Results
  • Data Migration Validation Reports
Investment: 41.5% of total ($805,000)
4

Implementation Partner & Optimization

Months 19-24

Full-time partnership through go-live and 30/90-day post-implementation reviews including system optimization, performance monitoring, and knowledge transfer to SMART team.

Key Milestones:

  • System Go-Live
  • 30-Day Post-Implementation Review
  • 90-Day Stabilization Complete
  • Knowledge Transfer to SMART Team
  • Final Acceptance & Sign-Off
Investment: 19.5% of total ($380,000)
